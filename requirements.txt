# ============================================================
# Installation Instructions for Android/Termux (Primary)
# ============================================================
# 1. Install Termux App:
#    - Download Termux from F-Droid or Google Play Store
#
# 2. Set Up Termux Environment:
#    pkg update && pkg upgrade
#    pkg install python
#    pkg install termux-api
#
# 3. Install Termux API APK:
#    - Download and install "Termux:API" APK separately
#    - Required for STT (speech-to-text) and TTS (text-to-speech)
#
# 4. Install Python Dependencies:
#    pip install -r requirements.txt
#
# 5. Run the Application:
#    python main.py
#
# ============================================================
# Notes:
# ============================================================
# - This codebase is optimized for Android/Termux
# - All dependencies have graceful fallbacks via safe import guards
# - Features will degrade gracefully if packages are unavailable
# - STT/TTS features require termux-api APK to be installed
# - If torch.cat() error on first run, re-run (known PyTorch initialization quirk)
# - Desktop fallbacks available for STT/TTS when not in Termux
